# SmartCare Hospital Management System

This is my project for a Hospital Management System. It is designed to manage various hospital operations and different user roles like admins, doctors, patients, receptionists, pharmacists, accountants, and lab technicians. 

The system allows patients to book appointments, doctors to manage their schedules, and the hospital staff to handle billing, lab reports, and pharmacy inventory.

## Technology Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## How to run the project locally

1. Open a terminal and go to the `backend` folder:
   ```
   cd backend
   npm install
   npm run dev
   ```

2. Open another terminal and go to the `frontend` folder:
   ```
   cd frontend
   npm install
   npm run dev
   ```

3. Open your browser and go to `http://localhost:5173` to view the website.
